# Steam 月度时长数据格式

可选文件：`content/steam/monthly_hours.json`

用于给任意年月补充真实的“本月时长”，键可以用 `appid`（推荐）或游戏名。

示例：

```json
{
  "2026-03": {
    "367520": 18.0,
    "Hollow Knight": 2.5
  },
  "2026-02": {
    "367520": 10.0
  }
}
```

说明：
- 页面选择某年月后，优先读取这个文件中的该月数据。
- 若该月没有配置：
  - 会显示为 0（不再使用 `playtime_2weeks` 近似）。
